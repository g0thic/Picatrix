# Picatrix
PICATRIX IS AN APP THAT PLAYS QURAN AUTOMATICALLY USING WINDOWS MEDIA PLAYER
WRITTEN WITH PYTHON 
TO MAKE THINGS WORK 
PLACE THE PICATRIX FOLDER IN THE "C:\" DRIVE   "C:\PICATRIX"
UNZIP THE C:\PICATRIX\AUDIO files
INSTALL PYTHON
RUN THE MAIN.PY USING COMMAND LINE (CMD)
